# <a name="the-windows-powershell-ise-scripting-object-model"></a>Windows PowerShell ISE Betik Oluşturma Nesne Modeli

Komut dosyası nesneleri, özelliklerini ve bunların yöntemleri özellikle bu amaç için tanımlanır.
