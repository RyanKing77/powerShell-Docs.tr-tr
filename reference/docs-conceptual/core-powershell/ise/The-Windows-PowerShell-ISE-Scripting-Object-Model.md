# <a name="the-windows-powershell-ise-scripting-object-model"></a>Windows PowerShell ISE nesne modeli komut dosyası oluşturma

Komut dosyası nesneleri, özelliklerini ve bunların yöntemleri özellikle bu amaç için tanımlanır.
